Aria2 ​

aria2 是一个轻量级的多协议和多源命令行下载实用程序。它支持 HTTP/HTTPS、FTP、SFTP、BitTorrent 和 Metalink。 aria2 可以通过内置的 JSON-RPC 和 XML-RPC 接口进行操作。

文档：https://aria2.github.io/manual/en/html/aria2c.html

github下载地址： https://github.com/aria2/aria2/releases/tag/release-1.37.0

https://github.com/aria2/aria2/releases/download/release-1.37.0/aria2-1.37.0.tar.gz

下载安装 ​

bash
apt install aria2

dnf install epel-release aria2

基本选项 ​

bash
-d          # 存放下载文件的目录
-o          # 下载文件的文件名。它始终相对于 -d 选项中给出的目录
-i          # 下载 FILE 中列出的 URI
-j          # 每个队列项目的最大并行下载数,默认值5
--all-proxy # 
--dry-run   # 仅检查远程文件是否可用，并不下载数据, 默认false
-m          # 设置尝试次数。 0 表示无限制, 默认值 5
-x          # 每次下载与一台服务器的最大连接数。默认值： 1
-t          # 设置超时（以秒为单位）。默认值： 60   
--conf-path # 默认值： $HOME/.aria2/aria2.conf,不存在则$XDG_CONFIG_HOME/aria2/aria2.conf
--no-conf   # 禁用加载 aria2.conf 文件
-D          # 作为守护进程运行。当前工作目录将更改为 / ，标准输入、标准输出和标准错误将重定向到 /dev/null 。默认值： false
--human-readable # 在控制台读数中以人类可读格式（例如 1.2Ki、3.4Mi）打印尺寸和速度。默认值： true
--log-level # 将日志级别设置为输出。 LEVEL 为 debug 、 info 、 notice 、 warn 或 error 。默认值： debug
--on-download-complete # 设置下载完成后执行的命令
--on-download-error # 设置由于错误而中止下载后要执行的命令
-q          # 没有控制台输出。默认值： false

使用示例 ​

bash
# 从网页下载
aria2c http://example.org/mylinux.iso

# 多个 来源
aria2c http://a/f.iso ftp://b/f.iso ***

# 主机使用 2 个连接进行下载
aria2c -x2 http://a/f.iso

# BitTorrent
aria2c http://example.org/mylinux.torrent

# BitTorrent Magnet URI
aria2c 'magnet:?xt=urn:btih:248D0A1CD08284299DE78D5C1ED359BB46717D8C'

# Metalink
aria2c http://example.org/mylinux.metalink

AriaNg ​

AriaNg 是一个现代 Web 前端，使 aria2 更易于使用。 AriaNg 是用纯 html 和 javascript 编写的，因此它不需要任何编译器或运行时环境。您只需将 AriaNg 放入您的网络服务器并在浏览器中打开它即可。 AriaNg 使用响应式布局，并支持任何桌面或移动设备

WebUI-Aria2 ​

bash
git clone https://github.com/ziahamza/webui-aria2.git /opt/webui-aria2

后台启动aria2 ​

vim ~/.config/aria2/aria2.conf

aria2c -D

bash

# 下载文件保存路径
dir=/opt/aria2-downloads

# 启用 JSON-RPC/XML-RPC 服务器, 默认:false
enable-rpc=true

# 接受所有远程请求, 默认:false
rpc-allow-origin-all=true

# 允许外部访问, 默认:false
rpc-listen-all=true

# RPC 监听端口, 默认:6800
rpc-listen-port=6800

# RPC 密钥
#rpc-secret=P3TERX

# RPC 最大请求大小
rpc-max-request-size=10M

# 断点续传
continue=true

# 始终尝试断点续传，无法断点续传则终止下载，默认：true
always-resume=false

# 会话文件保存路径
# Aria2 退出时或指定的时间间隔会保存`错误/未完成`的下载任务到会话文件
save-session=/root/.aria2/aria2.session

# 获取服务器文件时间，默认:false
remote-time=true

# 自定义 User Agent
user-agent=Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/93.0.4577.63 Safari/537.36 Edg/93.0.961.47
