zsh 编译安装最新版 ​

主页：https://www.zsh.org

下载页：https://www.zsh.org/pub

最新版本:https://www.zsh.org/pub/zsh-5.9.tar.xz

bash
yum install -y ncurses-devel gcc make
wget -nc https://www.zsh.org/pub/zsh-5.9.tar.xz -P /opt
cd /opt  && tar xf zsh-5.9.tar.xz && cd zsh-5.9 || exit
./configure --prefix=/opt/zsh # 默认/usr/local
make && make install

ln -s /opt/zsh/bin/zsh /usr/local/bin/zsh
#chsh -s /usr/local/bin/zsh  #CentOS 7
echo '/usr/local/bin/zsh' /etc/shells
usermod -s /usr/local/bin/zsh root   #CentOS 8
# 将zsh加入/etc/shells

主要的两个插件 ​

  • zsh-autosuggestions 自动提示你需要的命令
  • zsh-syntax-highlighting 命令行的命令高亮

Ubuntu ​

bash
apt install -y zsh zsh-autosuggestions zsh-syntax-highlighting
chsh -s /usr/bin/zsh

cat > ~/.zshrc << EOF
source /usr/share/zsh-autosuggestions/zsh-autosuggestions.zsh
source /usr/share/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh
EOF

source ~/.zshrc

自定义安装命令行工具及zsh ​

bash
#! /bin/bash
domain=10.0.0.10

cd /opt
# 下载命令行工具
wget -nc http://$domain/linux/bin/tools.tgz

[ ! -e /opt/tools ] && mkdir /opt/tools
tar xf tools.tgz -C tools

echo "PATH=/opt/tools:$PATH" > /etc/profile.d/tools.sh
[[ $PATH =~ 'tools' ]] || . /etc/profile

# zsh插件
autosuggest=http://$domain/linux/zsh-autosuggestions.tar.gz
high=http://$domain/linux/fast-syntax-highlighting.tar.gz

# zsh配置
zshrc=http://$domain/linux/.zshrc
aliasfile=http://$domain/linux/aliases.zsh

[ ! -e ~/.config/zsh ] && mkdir -p ~/.config/zsh
[ ! -e ~/.zsh ] && mkdir ~/.zsh

# zsh插件下载
cd ~/.zsh
wget -nc $autosuggest
wget -nc $high
tar xf zsh-autosuggestions.tar.gz -C ~/.zsh
tar xf fast-syntax-highlighting.tar.gz -C ~/.zsh

# zsh配置下载
wget -nc $zshrc -P ~
wget -nc $aliasfile -P ~/.config/zsh

echo "run source /etc/profile"

Bash揭示符 ​

Bash 提示符是通过环境变量 PS1 （提示符字符串 1Prompt String 1） 来设置的，它用于交互式 shell 提示符。

如果你需要更多的输入才能完成一个 Bash 命令时，PS2 环境变量就是用来设置多行提示符的

  • \u：用户名

  • \h：短主机名

  • \W：当前你所在的目录的名称（basename），~ 表示你的主目录

  • \s：Shell 名字（bash 或者 sh，取决于你的 Shell 的名字是什么）

  • \v：Shell 的版本号

  • \d：将日期扩展成 “Tue Jun 27” 这种格式

  • \D{fmt}：允许自定义日期格式——可通过 man strftime 来获得更多信息

  • \D{%c}：获得本地化的日期和时间

  • \n：换行（参考下面的多行提示符）

  • \w：显示当前工作目录的完整路径

  • \H：当前工作机器的完整主机名

zsh补全 ​

bash
# 要初始化当前会话的补全，您需要自动加载函数 compinit 并调用它,在文件 .zshrc 中添加
autoload -U compinit; compinit

# 使用 zstyle 配置 Zsh 模块的一般方法
zstyle <pattern> <style> <values>

# <pattern> 
:completion:<function>:<completer>:<command>:<argument>:<tag>

# 示例
# 输入 cp 后点击 TAB ，补全系统匹配的可能文件将按大小排序
zstyle ':completion:*:*:cp:*' file-sort size
# 模式 :completion:*:*:cp:* 优先于 :completion:* ，因为它被认为更精确
zstyle ':completion:*' file-sort modification

# 可以创建一个新文件 completion.zsh 并直接在文件 .zshrc 中获取它
# 也要直接写在.zshrc中
source /path/to/my/completion.zsh
