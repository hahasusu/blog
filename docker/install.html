Docker 安装 ​

1. 脚本安装 ​

在测试和开发环境中，您可以使用自动化便捷脚本来安装 Docker。

安装脚本需要 root 或 sudo 权限才能安装和使用 Docker

bash
curl -fsSL https://get.docker.com -o get-docker.sh

sh get-docker.sh


# docker 服务在基于 Debian 的发行版上自动启动。
# 在基于 RPM 的发行版（例如 CentOS、Fedora、RHEL 或 SLES）上，您需要使用适当的 systemctl 或 service 命令手动启动它
# 默认情况下非 root 用户无法运行 Docker 命令

2. rpm 仓库 ​

阿里云各个发行版仓库 https://mirrors.aliyun.com/docker-ce/linux/

bash
# yum install -y yum-utils
# yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
wget https://mirrors.aliyun.com/docker-ce/linux/rhel/docker-ce.repo -O /etc/yum.repos.d/docker-ce.repo
dnf makecache

# 最新版
yum install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

# 指定版本
yum list docker-ce --showduplicates | sort -r
yum install docker-ce-VERSION docker-ce-cli-VERSION containerd.io docker-buildx-plugin docker-compose-plugin

systemctl start docker
docker run hello-world

多个镜像源 ​

bash
cat > /etc/yum.repos.d/docker-ce.repo <<EOF
[docker-ce-stable]
name=Docker CE Stable
baseurl=https://mirrors.ustc.edu.cn/docker-ce/linux/rhel/8/x86_64/stable/
        https://mirrors.huaweicloud.com/docker-ce/linux/rhel/8/x86_64/stable/
        https://mirrors.aliyun.com/docker-ce/linux/rhel/8/x86_64/stable/
gpgcheck=0
EOF

# dnf install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
# 会自动安装其它依赖
dnf install -y docker-ce

配置守护进程 ​

https://docs.docker.com/config/daemon/

代理设置1 ​

bash
mkdir -p /lib/systemd/system/docker.service.d
touch /etc/systemd/system/docker.service.d/http-proxy.conf

[Service]
Environment="HTTP_PROXY=http://proxy.example.com:3128"
Environment="HTTPS_PROXY=https://proxy.example.com:3129"
Environment="NO_PROXY=10.0.0.0/8"

systemctl daemon-reload
systemctl restart docker

代理设置2 ​

在 Docker Engine 23.0 及更高版本中，您还可以在 daemon.json 文件中为守护进程配置代理行为

这些配置会覆盖默认的 docker.service systemd 文件

json
{
  "proxies": {
    "http-proxy": "http://192.168.216.1:7890",
    "https-proxy": "https://192.168.216.1:7890",
    "no-proxy": "*.test.example.com,.example.org,127.0.0.0/8"
  }
}
