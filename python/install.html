Python 安装 ​

官网：https://www.python.org/

仓库安装 ​

bash
dnf install -y python3.11 python3.11-pip
1

源码编译安装 ​

https://www.python.org/downloads/source/

依赖：https://devguide.python.org/getting-started/setup-building/#install-dependencies

  • zlib：用于支持压缩和解压缩。
  • libffi：用于支持调用外部函数。
  • libssl-dev：用于支持SSL/TLS加密功能。
  • libsqlite3-dev：用于支持SQLite数据库。
  • libbz2-dev：用于支持bzip2压缩。
  • libreadline-dev：用于支持命令行编辑。
  • libncurses-dev：用于支持文本界面显示。
  • pkg-config 是一个用于检查和配置软件包的工具，它通常用于自动检测依赖项的路径和版本信息。
bash
wget https://www.python.org/ftp/python/3.12.0/Python-3.12.0.tar.xz
tar xvf Python-3.12.tar.xz
cd Python-3.12.0
./configure --prefix=$HOME/.local --enable-optimizations
make -j4
make install

# 测试
python -V

PyPI 镜像源 ​

临时使用

python
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple some-package
1

设为默认

python
# 升级 pip 到最新的版本
# python -m pip install --upgrade pip
python -m pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --upgrade pip

pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

# 查看
pip config list
